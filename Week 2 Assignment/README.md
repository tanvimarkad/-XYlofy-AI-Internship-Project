# Employee Attrition Prediction
 Project Overview
This project predicts whether an employee is likely to leave the company using Machine Learning.
## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn
- Jupyter Notebook
## Models Used
- Logistic Regression
- Random Forest
- Gradient Boosting
## Evaluation Metrics
- Precision
- Recall
- F1 Score
- ROC-AUC
- Confusion Matrix
## Outcome
The best-performing model identifies key factors contributing to employee attrition and provides actionable insights to support HR retention strategies.
